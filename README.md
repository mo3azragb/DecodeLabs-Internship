# DecodeLabs Internship - AI Project

Welcome to my repository for the DecodeLabs Internship. This folder contains all the projects completed during the training period.

## 📁 Projects Overview

### 1. Simple Chatbot (`bot.py`)
- **Description:** A rule-based AI chatbot that interacts with users and responds to specific keywords.
- **How to run:** `python bot.py`

### 2. Iris Flower Classification (`classification_project.py`)
- **Description:** A Machine Learning model using the KNN algorithm to classify Iris flower species based on their features, achieving an accuracy of over 95%.
- **How to run:** `python classification_project.py`

### 3. Store Recommendation System (`recommendation_system.py`)
- **Description:** An AI-driven recommendation logic that analyzes store sales from an Excel dataset (`Online-Store-Orders.xlsx`) and recommends the top 3 best-selling products based on user input.
- **How to run:** `python recommendation_system.py`
